
Simple Library Application made with Laravel 5.3 and Vue.js 2.0

Instructions:
	- composer install
	- create database
	- edit .env.example file with your db name, db username, db pass and save it as .env file
	- from your project root directory run the following commands:
		php artisan key:generate
		php artisan migrate --seed
